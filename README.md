# Sistema de Exploración Neuroambiental

Este proyecto fue desarrollado para el curso de Informática II de Bioingeniería. Permite analizar datos de calidad del aire (SIATA) y señales EEG en formato .mat.

## Funcionalidades

* Lectura de archivos CSV (SIATA)
* Visualización de datos (plot, boxplot, histograma)
* Operaciones con apply, map y suma de columnas
* Remuestreo de datos (diario, mensual, trimestral)
* Procesamiento de señales EEG
* Suma de canales
* Cálculo de promedio y desviación estándar

## Cómo ejecutar

1. Abrir la terminal en la carpeta del proyecto
2. Ejecutar:
   py main.py
3. Seguir el menú en pantalla
